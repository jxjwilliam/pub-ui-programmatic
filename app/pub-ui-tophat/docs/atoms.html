Buttons

Button styles can be applied to either <button> or <a>:

To do this Use this
Perform an action, such as submit a form or trigger a JS event <button>
Link to another page <a>

Button 
  <a class="button default">Button</a>
  <button type="button default">Button</button>

Button Colors

Button colors correspond to status colors. Use good judgement when using a colored button. Is it of importance to call attention to? We don't want the entire interface to be noisy and distract the user from the information and task at hand.

Button Class Description
button This this is the go to button style
button primary-bg Use this button as the primary call to action
button secondary-bg Use this as a secondary action button
button alert-bg Use this for alerts and critical events
button warning-bg Use this for warnings and major events
button yield-bg Use this for yields and caution
button success-bg Use this for confirmations and successful events
button inverse-bg Use this for inversion coersion
button disabled This button is deactivated 

  <button class="button primary-bg">Signup now</button>
  <button class="button secondary-bg">Cancel</button>
  <button class="button alert-bg">Delete</button>
  <button class="button warning-bg">Set override</button>
  <button class="button yield-bg">Throttle</button>
  <button class="button success-bg">Success</button>
  <button class="button inverse-bg">Clear</button>
  <button class="button disabled">Activate</button>

Invisible buttons

There are also invisible buttons. Use these when you want to keep a button in view but want to deemphasize its importance.

Button Class Description
button invisible Use this to deemphasize its' importance
button invisible alert-bg Deemphasize its importance, but warn of its' danger 

    <button class="button invisible">Invisible</button>
    <button class="button invisible alert-bg">Invisible</button>

Fonts

atoms/global/_fonts.scss
last updated 8/31/14

The official brand font is Helvetica Neue LT, which works well in print. However, for on screen and in our app, we use Open Sans as the system font for it's more humanistic feel. Designed specifically for screens and interfaces, it's open letterforms allow for better readability. We use Lato for numeric values for it's high contrast and fresh, modern and timeless feel. This gives our data a distinct and strong voice.

There are two ways to use call our typefaces: CSS classes and Sass variables.

Type Name
CSS class .primary-font, .data-font
Sass variable $primary-font, $data-font

Scrollbars

atoms/global/_scrollbars.scss
last updated: 8/31/14

To add Pubalicious scrollbars, simply define .scroll on a container <div>. The container should have a fixed dimension, and overflow set to auto or scroll for that axis. For instance, set a fixed height and overflow-y to scroll or auto. Lastly, place an object in the container that is larger.

    <div class="scroll" style="height:200px;width:262px;overflow-y:scroll;overflow-x:hidden">
        <div>
            <img src="http://bit.ly/1iBLqvd" />
        </div>
    </div>

Drag & Drops

If you want to do dynamic drag-and-drops from a data-table, add .drag-and-drop-move to the <tr> being dragged.

Draaaaaagg me 
<table>
  <tbody>

Data Table List

atoms/tables/_data-table-list.scss
last updated 8/31/14 –

Use a data table list when you need lists, and not tabular data. When showing large amounts of data, structure your data, and prioritize what you want the user to focus on first. Here, less important information (Date Saved) is de-emphasized through smaller header font size and grey text.

On a list, we can also embed buttons on hover. Simply add a button to a cell, and the hover with automagically occur.

Your saved reports

Nike Revenue Report August 26 @ 12:46am PST
Revenue by Advertiser Revenue, impressions and eCPM shown by Advertiser
