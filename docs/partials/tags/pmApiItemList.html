pmApiItemList

Description

Call any PubMatic API and display the results in a UX compliant Item List. (originally designed for Package Screen)

Usage

as element:
<pm-api-item-list
       api-endpoint="{string}"
       method="{string}"
       field-list="{string}">
</pm-api-item-list>

Parameters

ParamTypeDetails
apiEndpointstring

PubMatic API Endpoint (REST) called to populate the data table.

methodstring

API CRUD method used for call (GET, POST, PUT, PATCH)

fieldListstring

Comma delimited list fields returned by the API that will display in the data table.

Example

Call Inventory API and display XYZ in the data table. 

   <pm-api-item-list
      endpoint="commons/packages"
      method="GET"
      displayfields="api,endpoint,method,description">
   </pm-api-item-list>
