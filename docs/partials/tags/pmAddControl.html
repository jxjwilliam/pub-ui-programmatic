pmAddControl

Description

Configurable Add button control for adding any PubMatic object (products, packages etc.)

Usage

as element:
<pm-add-control
       controlobjects="{string}">
</pm-add-control>

Parameters

ParamTypeDetails
controlobjectsstring

List of JSON objects that can be added

Example

Configure Add button to enable addition of products 

    <pm-add-control
       controlobjects='[{"id":"create-product-choice","toolTip":"Create a Product","icon":"fa-cube","templateUrl":""}'
    </pm-add-control>
