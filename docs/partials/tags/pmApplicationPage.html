pmApplicationPage

Description

Add a CRUD (Create, Read, Update Delete) Application Page to any PubMatic REST API

Usage

as element:
<pm-application-page
       title="{string}"
       subtitle="{string}"
       description="{string}">
</pm-application-page>

Parameters

ParamTypeDetails
titlestring

Main title

subtitlestring

Page subtitle

descriptionstring

Page description

Example

Generate a brand new Inventory Discovery page with 2 HTML tags 

    <pm-application-page
        title="My First Page"
        sub-title="Subtitle for my first page"
        description="This is a simple page I created using a single HTML tag.">

        My Page Content Here!

    </pm-application-page>
