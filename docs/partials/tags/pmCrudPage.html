pmCrudPage

Description

Add a page to your web application with a single HTML tag

Usage

as element:
<pm-crud-page
       title="{string}"
       subtitle="{string}"
       description="{string}"
       acl-write-access-string="{string}"
       acl-bulk-write-access-string="{string}"
       page-size="{string}">
</pm-crud-page>

Parameters

ParamTypeDetails
titlestring

Main title

subtitlestring

Page subtitle

descriptionstring

Page description

acl-write-access-stringstring

ACL access string determining whether user is read only

acl-bulk-write-access-stringstring

ACL access string used to determining whether users can bulk load/update

page-sizestring

Number of items to show per page

Example

Generate a brand new Inventory Discovery page with 2 HTML tags 

<pm-crud-page
    title="Products Screen"
    sub-title="View or Add Products"
    description="Select a Product below to edit it or add a new product."
    acl-write-access-string="castrum-product-write"
    acl-bulk-write-access-string="castrum-product-bulk-write"
    restendpoint="product"
    listfields="name,description,category,position,sites">
</pm-crud-page>
