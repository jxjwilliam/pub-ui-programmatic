pmApiTable

Description

Call any PubMatic API and display the results in a UX compliant HTML table.

Usage

as element:
<pm-api-table
       api-endpoint="{string}"
       method="{string}"
       field-list="{string}">
</pm-api-table>

Parameters

ParamTypeDetails
apiEndpointstring

PubMatic API Endpoint (REST) called to populate the data table.

methodstring

API CRUD method used for call (GET, POST, PUT, PATCH)

fieldListstring

Comma delimited list fields returned by the API that will display in the data table.

Example

Call Inventory API and display XYZ in the data table. 

   <pm-api-table
      endpoint="inventory"
      method="GET"
      displayfields="api,endpoint,method,description">
   </pm-api-table>
